## 🧠 About
**TBFM (Tkinter Based Form Maker)** is a Python tool that helps you design GUI forms visually using Tkinter.  
It’s great for quickly building windows and generating the Python code behind them.

---

## 🚀 Features
- Create Tkinter windows and forms easily  
- Simple, intuitive interface    
- Custom icons

---

## 🧰 Requirements
- **Python 3.x**  
- **Tkinter** (included with most Python installations)  

---

To run the program, open a terminal or command prompt and type:
