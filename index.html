TBFM - Tkinter Based Form Maker

About TBFM

TBFM (Tkinter Based Form Maker) is a lightweight and simple GUI tool for creating Python forms using the Tkinter library. Designed by Caron Software, it allows you to easily make form-based windows without coding everything manually!

Features

Easy to Use

Simple interface that anyone can use to make forms.

Custom Icons

Set custom icons for your forms easily (TBFM.ico).

About Menu

See app info quickly using the built-in About menu.
