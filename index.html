TBFM

Tkinter Based Form Maker

About TBFM

TBFM (Tkinter Based Form Maker) is a Python application created by Caron. It allows you to quickly make custom forms using a simple graphical interface built with Tkinter.

Features

  • Create custom forms with multiple input types.
  • Set window sizes and options.
  • Easy to use interface.
  • Completely free and open-source.
