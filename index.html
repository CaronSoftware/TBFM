About TBFM

TBFM is a lightweight and simple GUI tool for creating Python forms using Tkinter.
Designed by Caron Software, it allows you to easily make form-based windows without writing code!

Features

Easy to Use

Simple interface that anyone can use to make forms.

Fully written in Python

TBFM Uses python to MAKE python. Wow.
