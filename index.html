TBFM

Tkinter Based Form Maker

About TBFM

TBFM (Tkinter Based Form Maker) is a Python tool that lets you visually design GUI forms using Tkinter. It's perfect for quickly creating windows and generating Python code for your forms.

Features

Easy to Use

Simple interface that anyone can use to make forms.

Custom Icons

Set custom icons for your forms easily (TBFM.ico).

About Menu

See app info quickly using the built-in About menu.
